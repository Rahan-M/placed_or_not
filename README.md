# placed_or_not
A project to predict whether a student will be placed or not.
I train an XGBoost classifier model to predict whether a student will be placed or not using the following features
- StudentID
- CGPA
- Internships
- Projects
- Workshops/Certifications
- AptitudeTestScore
- SoftSkillsRating
- ExtracurricularActivities
- PlacementTraining
- SSC_Marks,
- HSC_Marks, 
- PlacementStatus
